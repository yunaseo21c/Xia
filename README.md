# 시아(Xia)
✨️ 여러분의 서버를 지키고 도와주는 시아에요 !

---

## 시아가 뭐에요 ?
시아는 디스코드 유틸리티 봇으로,
서버를 관리하고 사용자들에게 다양한 기능을 제공하는 서비스에요.

미니게임, 서버 관리 기능 등을 포함해
서버에 필수적인 기능들을 제공하는 서비스에요.

별을 찾아 떠나는 시아의 여정은,
모든 서버를 책임지고 지킬 때까지 이어질 예정이에요.

---

## 기능
> [!NOTE]
> 아직 완벽하게 변경되지 않았음으로 작성하지 아니하였습니다.
> 추후 기능 개발 후 작성될 예정입니다.

## 버그 제보 및 수정
버그 제보는 [Team Starlight 서포트서버](https://discord.gg/twwuHcpfft) 또는 저장소 이슈탭에 등록하여 주세요.
버그 제보 시 하단과 같은 양식을 지켜주시기 바랍니다.

```
오류 발생 시각 : 
표시된 오류 메시지 :
재현 방법 :
```

아래는 예시입니다.
```
오류 발생 시각 : 2026년 5월 20일 11시 3분
표시된 오류 메시지 : syntex error ...
재현 방법 : 
1. /로그 명령어 실행
2. 로그 카테고리 지정
```

## 기능 추가 요청
기능 추가를 요청 할 경우,
저장소 이슈탭에서 요청하여 주세요.

기능 요청 시 자세하게 작성하여 주시기를 바라며,
요청된 내용에 대하여는 작성자의 요구에 완벽하게 부합할 수는 없음을 양지해주세요.
